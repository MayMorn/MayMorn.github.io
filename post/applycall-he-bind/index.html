apply、call和bind

2020-11-23 javascript

apply

定义

apply()方法调用一个函数，其具有一个指定的this值，以及作为一个数组(或类似数组的对象)提供的参数。

语法

func.apply(thisArg, [argsArray])

  • thisArg：在func函数运行时指定的this值。如果函数处于非严格模式下，则指定为null或者undefined时会自动指向全局对象(浏览器中就是window对象)，同时值为原始值(数字，字符串，布尔值)的this会指向该原始值的自动包装对象。
  • argsArray：一个数组或者类数组对象，其中的数组元素将作为单独的参数传给func函数。如果该值为null或undefined则表示不需要传入任何参数。
const numbers = [5, 6, 7, 2, 1]

const max = Math.max.apply(null, numbers)

console(max) // 7

call

call与apply类似，区别在于call方法接受的是参数列表，而apply()方法接受的是一个参数数组。

function Product(name, price) {
    this.name = name
    this.price = price
}
function Food(name, price) {
    product.call(this, name, price)
    this.category = 'food'
}
console.log(new Food('cheese', 5).name) // cheese

bind

定义

bind方法创建一个新的函数，在bind被调用时，这个新函数的this被指定为bind的第一个参数，而其余参数将作为新函数的参数，供调用时使用。

const module = {
    x: 42,
    getX: function() {
        return this.x
    }
}

const unboundGetX = module.getX
console.log(unboundGetX()) // undefined

const boundGetX = unboundGetX.bind(module)
console.log(boundGetX) // 42

总结

  • apply，call，bind都可以用来改变函数的this对象的指向。
  • 三者的第一个参数都是this要指向的对象，也就是想指定的上下文。
  • bind返回一个函数以便于稍后调用，而apply，call则是立即调用。
