记录几个面试/笔试题

2020-11-10

1

下面代码的输出结果是什么

var val = 12
function func1() {
    console.log(val)
    var val = 20
    console.log(val)
}

答案： undefined 20
知识点：函数会优先找自己的内部变量，var变量提升，所以第一个val为undefined

2

创建带有ID属性的DOM元素有什么副作用？
解答：如果一个元素拥有ID属性，那么ID属性的属性值就会成为window对象的属性名。

3

以下代码执行后，输出什么？

var a = 10
function a() {}
console.log(typeof a)

答案：number
知识点：函数提升优先级高于变量提升，所以代码等价于

function a() {}
var a;
a = 10
console.log(typeof a)

4

举例几个不支持冒泡的事件
答案：

  • focus
  • blur
  • mouseenter
  • mouseleave
  • load
  • unload
  • resize

5

下面的代码中有几个变量没有被回收

var i = 1;
var i = 2;
var add = function() {
    var i = 0;
    return function() {
        i++;
        console.log(i)
    }
}
add()

答案：3个
解析：

  • 全局变量不会被回收
  • 局部变量会被回收，也就是函数一旦运行完以后，函数内部的东西会被销毁
  • 只要被另一个作用域所引用就不会被回收
